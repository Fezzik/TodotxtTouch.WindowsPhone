todo.txt for Windows Phone

Future-proof task management for your Windows Phone

todo.txt for Windows Phone is a task/notes manager based on the todo.txt file format created by Gina Trapani. The todo.txt file format is a simple text file format that's usable on any platform (because it's just text). The application can be used as a stand-alone manager for your tasks, but the real magic comes when you use Dropbox to synchronize your todo.txt file across all of your computers and devices. The app is available for free on the Windows Phone Marketplace.
The current version supports:
  • Syncing with Dropbox
  • Archiving completed tasks (to done.txt)
  • Filtering tasks by +project and @context
  • Setting priority tasks
Targeted features for future versions:
  • Phone call integration
  • Email integration
  • Task search
  • Web browser integration
  • SMS integration
